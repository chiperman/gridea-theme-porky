## 🔧安装方法

方法一：

- 克隆项目到本地
- 将「porky」文件夹移动到到博客源文件夹「themes」内
- 重启「 Gridea 」客户端
- 在主题中选择「 porky」主题，点击保存
- 预览查看效果

方法二：

```
# 安装方法
$ cd your-blog/themes
$ git clone https://github.com/chiperman/gridea-theme-porky.git

# 更新方法
git pull origin master
```

## 💡特色

- 支持 valine 评论
- 支持社交功能
- 支持亮色 / 暗色模式自动切换
- 支持移动端页面适配
